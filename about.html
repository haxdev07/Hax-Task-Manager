Project Overview

This is a simple Task Manager application designed to help users create, organize, and track their daily tasks. The application works completely offline using the browser's local storage, so tasks remain saved even after closing the window.

Key Features

  • Add, edit, delete tasks
  • Mark tasks as completed or pending
  • Priority levels: Low, Medium, High
  • Due date support and overdue highlighting
  • Search tasks by title or description
  • Sort tasks by creation date, due date, or priority
  • Separate pages for Dashboard, Pending tasks, and Completed tasks
  • Dark / Light theme toggle stored in browser
  • Fully offline support using browser localStorage

How It Works (Short Explanation)

  1. When the user adds a task, it is stored in an in-memory array.
  2. The task list is saved to the browser's localStorage.
  3. When the page loads, tasks are read back from localStorage.
  4. JavaScript dynamically creates HTML elements to display each task.
  5. Filters (pending/completed), search, and sort are all applied in JavaScript.

Future Improvements

  • Add task categories or labels for better organization
